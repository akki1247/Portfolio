project

  • Labour-supply-chain

    I developed a Labor Supply Chain Platform using HTML, CSS, React, Express, and MySQL database to bridge the gap between farmers and laborers, offering a unique solution tailored for the agricultural sector and other non-technical fields. Unlike traditional job portals that focus on IT, sales, or similar industries, this platform is designed to cater to the hiring needs of farmers and contractors. Using Express, I built APIs to handle key functionalities such as user registration, profile creation, and job listings. Each laborer can create a comprehensive profile, including personal details and a video showcasing their skills. This feature enables farmers and contractors to make informed decisions when hiring, based on the laborer's work history and capabilities. The platform supports both individual laborers and contract-based labor groups, providing a centralized hub where farmers can easily find and hire laborers according to their specific requirements. This project aims to revolutionize the hiring process in the agricultural sector, making it more efficient and transparent.

    I developed an Order Cart Handling System using React, HTML, and CSS to provide a seamless shopping experience.
